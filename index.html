Ajuda

Como abrir um chamado

  1. Selecione o tipo de chamado (Planilha, Vídeo ou Outro)
  2. Preencha as informações solicitadas
  3. Para vídeos, selecione os arquivos na ordem correta
  4. Defina um prazo para conclusão
  5. Clique em "Enviar Chamado"

Tipos de Chamado

  • Planilha: Para problemas ou solicitações relacionadas a planilhas
  • Vídeo: Para envio e edição de vídeos (é possível enviar múltiplos arquivos)
  • Outro: Para qualquer outro tipo de solicitação

Níveis de Urgência

O sistema calcula automaticamente a urgência com base no prazo definido:

  • Baixa: Prazo maior que 7 dias
  • Média: Prazo entre 3 e 7 dias
  • Alta: Prazo menor que 3 dias
